JavaScript App

(Pokédex)


Project Description

A small web application developed using HTML, CSS, and JavaScript that loads data from an external API. The user is able to view the data in more detail on demand.

How to use the project

Browse or search for Pokémon, interact with a card to display more detailed information about the selected Pokémon in a modal. All information is loaded from an external API.
